Limite del Valor Absoluto & Funcion Escalonada de Heaviside

Autor: Ing. Stivenson Rincon Tutor: Ing. Antonio

📚 Introduccion

⚙️ Imagina esto: Un ingeniero fabrica un eje que debe medir exactamente 5.00 cm. La pieza real mide 4.97 cm. ¿Esta bien o esta mal? La respuesta depende de la tolerancia: si el error permitido es 0.05 cm, calculamos |5.00 − 4.97| = 0.03 < 0.05... ¡la pieza es aceptable! Este concepto de «distancia sin importar la direccion» es el valor absoluto, y entender como se comportan sus limites es clave en calculo diferencial.

📖 Teoria

🔹 Definicion de Valor Absoluto
|x| = { xsi x ≥ 0 −xsi x < 0

Geometricamente, |x| es la distancia del numero x al origen en la recta numerica. Siempre es un valor no negativo.

💡 Teorema: Limites con Valor Absoluto

Para evaluar limx→a |f(x)| debemos analizar los limites laterales:

limx→a− |f(x)|    y    limx→a+ |f(x)|

El limite existe si y solo si ambos limites laterales son iguales.

⚠️ Clave: Cuando tenemos |f(x)| / f(x), debemos separar en casos segun el signo de f(x). Si f(x) > 0, el cociente es 1. Si f(x) < 0, el cociente es −1. ¡Los limites laterales seran distintos!

🔎 Metodo paso a paso

  1. Identificar el punto critico (donde la expresion dentro de |·| cambia de signo).
  2. Calcular el limite por la izquierda (x → a−): sustituir |f(x)| por −f(x) cuando f(x) < 0.
  3. Calcular el limite por la derecha (x → a+): sustituir |f(x)| por f(x) cuando f(x) > 0.
  4. Comparar: si son iguales, el limite existe. Si no, el limite no existe.

🔬 Demostracion del Teorema

Teorema: Existencia del Limite
limx→a f(x) = L   si y solo si   limx→a− f(x) = L = limx→a+ f(x)

Verifiquemos este teorema evaluando f(x) = |x| y f(x) = |x|/x en el punto critico x = 0. Haz clic en el boton para revelar cada verificacion.

✅ Caso 1: f(x) = |x| — Funcion continua

Evaluacion Calculo Resultado
Forma de f(x) |x| = x si x≥0, −x si x<0 Funcion «V»
Dominio y Rango Dom: ℝ   Rango: [0, +∞) Definida en todo ℝ
Lim. Izquierda limx→0− |x| = limx→0− (−x) = 0 0
Lim. Derecha limx→0+ |x| = limx→0+ x = 0 0
¿Coinciden? 0 = 0   ✅ LIMITE EXISTE = 0
✅ Teorema verificado: Como limx→0− |x| = 0 = limx→0+ |x|, el teorema confirma que el limite existe y vale 0. La funcion |x| es continua en x = 0 a pesar de tener un «pico».

❌ Caso 2: f(x) = |x|/x — Funcion con salto

Evaluacion Calculo Resultado
Forma de f(x) |x|/x = 1 si x>0, −1 si x<0 Dos lineas horizontales
Dominio y Rango Dom: ℝ − {0}   Rango: {−1, 1} No definida en x=0
Lim. Izquierda limx→0− |x|x = −xx = −1 −1
Lim. Derecha limx→0+ |x|x = xx = 1 1
¿Coinciden? −1 ≠ 1   ❌ LIMITE NO EXISTE
❌ Teorema verificado: Como limx→0− |x|/x = −1 ≠ 1 = limx→0+ |x|/x, el teorema confirma que el limite no existe. Hay una discontinuidad de salto de magnitud 2 en x = 0.

✏️ Ejemplos Resueltos

Ejemplo 1:  limx→0 |x|x ▼
Paso 1 — Identificar punto critico
El denominador es x y el argumento del valor absoluto es x. El punto critico es x = 0.
Paso 2 — Limite por la izquierda
Cuando x → 0−, tenemos x < 0, entonces |x| = −x:
|x|x = −xx = −1
Paso 3 — Limite por la derecha
Cuando x → 0+, tenemos x > 0, entonces |x| = x:
|x|x = xx = 1
❌ Conclusion
Como −1 ≠ 1, los limites laterales son diferentes. El limite NO existe.
Ejemplo 2:  limx→3 |x−3|x−3 ▼
Paso 1 — Identificar punto critico
La expresion dentro del valor absoluto es x − 3. Cambia de signo en x = 3.
Paso 2 — Limite por la izquierda
Cuando x → 3−, tenemos x < 3, entonces x−3 < 0 y |x−3| = −(x−3):
−(x−3)x−3 = −1
Paso 3 — Limite por la derecha
Cuando x → 3+, tenemos x > 3, entonces x−3 > 0 y |x−3| = x−3:
x−3x−3 = 1
❌ Conclusion
−1 ≠ 1 ⇒ El limite NO existe.
Ejemplo 3:  limx→1 |x − 1| ▼
Paso 1 — Identificar la funcion
Aqui solo tenemos |x − 1| sin division. La funcion valor absoluto es continua en todo punto.
Paso 2 — Sustitucion directa
Como |x−1| es continua, podemos sustituir directamente:
|1 − 1| = |0| = 0
Paso 3 — Verificacion con limites laterales
Izquierda: |x−1| = −(x−1) = 1−x → 0
Derecha: |x−1| = x−1 → 0
Ambos dan 0. ✅
✅ Conclusion
Los limites laterales coinciden. El limite EXISTE y vale 0.

🎯 Practica Interactiva

Puntaje: 0 / 3
1. Evalua:  limx→0 |2x|x
✅ Correcto! Por la izquierda: |2x|/x = −2x/x = −2. Por la derecha: |2x|/x = 2x/x = 2. Como −2 ≠ 2, el limite no existe.
❌ Incorrecto. Pista: Analiza los limites laterales. ¿Que pasa con |2x| cuando x < 0?
💡 Solucion: Cuando x → 0−: |2x|/x = −2x/x = −2. Cuando x → 0+: |2x|/x = 2x/x = 2. Como −2 ≠ 2, el limite no existe.
2. Evalua:  limx→5 |x−5|x−5
✅ Correcto! Los limites laterales son −1 (izquierda) y 1 (derecha), por lo que el limite no existe.
❌ Incorrecto. Pista: Es la misma estructura que |x|/x pero desplazada a x=5.
💡 Solucion: Por la izquierda (x<5): −(x−5)/(x−5) = −1. Por la derecha (x>5): (x−5)/(x−5) = 1. Como −1 ≠ 1, no existe.
3. Evalua:  limx→1 |x − 1|
✅ Correcto! |x−1| es continua, asi que el limite es simplemente |1−1| = 0.
❌ Incorrecto. Pista: No hay division aqui. La funcion |x−1| es continua. Solo sustituye x=1.
💡 Solucion: La funcion |x−1| es continua en todo punto. Sustituyendo: |1−1| = |0| = 0.

🌎 Aplicaciones en el Mundo Real

⚙️

Tolerancias de fabricacion

En ingenieria, la calidad de una pieza se mide comparando la medida real con la ideal usando valor absoluto.

|medidareal − medidaideal| < tolerancia
📡

Rectificacion de senales

En electronica, los circuitos rectificadores usan el principio del valor absoluto para convertir corriente alterna (AC) en corriente continua (DC).

VDC = |VAC(t)|
🎯

Analisis de errores

El error absoluto mide la precision de una medicion sin importar si el error fue por exceso o por defecto.

Error = |valormedido − valorreal|

📚 Introduccion

💡 Imagina un interruptor de luz: esta apagado... y de pronto, encendido. No hay transicion gradual, no hay dimmer. Este comportamiento «todo o nada» es exactamente lo que modela la funcion escalon de Heaviside. Fue creada por Oliver Heaviside (1850-1925), un ingeniero electrico autodidacta britanico que revoluciono el analisis de circuitos. Hoy su funcion es fundamental en ingenieria, fisica y procesamiento de senales.

📖 Teoria

🔹 Definicion de la Funcion Escalon de Heaviside
H(t) = { 0si t < 0 1si t ≥ 0

Es una funcion que «enciende» un valor en un instante especifico. Antes del instante: 0 (apagado). Despues: 1 (encendido).

🔹 Version Desplazada
H(t − a) = { 0si t < a 1si t ≥ a

Desplazar por a mueve el «encendido» al instante t = a.

💡 Propiedad clave: Discontinuidad de salto
limt→0− H(t) = 0      limt→0+ H(t) = 1

Como 0 ≠ 1, el limite en t = 0 NO existe. Esto se llama discontinuidad de salto con magnitud 1.

🔗 Relacion con el Valor Absoluto
H(x) = x + |x|2x    para x ≠ 0

Esta formula conecta ambos temas de esta OVA: el valor absoluto y la funcion escalon estan intimamente relacionados.

⚠️ Funciones compuestas con Heaviside: Podemos construir cualquier funcion escalonada (como una escalera) sumando multiples funciones de Heaviside desplazadas. Por ejemplo: f(t) = H(t) + H(t−1) + H(t−2) crea una escalera de 3 peldanos.

🔬 Demostracion del Teorema

Teorema: Existencia del Limite
limt→a f(t) = L   si y solo si   limt→a− f(t) = L = limt→a+ f(t)

Verifiquemos este teorema evaluando la funcion escalon de Heaviside H(t) en distintos puntos. Haz clic para revelar cada verificacion.

❌ Caso 1: H(t) en t = 0 — Punto de discontinuidad

Evaluacion Calculo Resultado
Forma de H(t) H(t) = 0 si t<0,   1 si t≥0 Escalon en t = 0
Dominio y Rango Dom: ℝ   Rango: {0, 1} Solo dos valores posibles
Lim. Izquierda limt→0− H(t) = 0   (zona «apagada») 0
Lim. Derecha limt→0+ H(t) = 1   (zona «encendida») 1
¿Coinciden? 0 ≠ 1   ❌ LIMITE NO EXISTE
❌ Teorema verificado: Como limt→0− H(t) = 0 ≠ 1 = limt→0+ H(t), el teorema confirma que el limite no existe en t = 0. Hay un salto de magnitud 1: la funcion «salta» instantaneamente de apagado a encendido.

✅ Caso 2: H(t) en t = 2 — Punto de continuidad

Evaluacion Calculo Resultado
Punto t = 2 Como 2 > 0, estamos en la zona donde H(t) = 1 Zona constante
Lim. Izquierda limt→2− H(t) = 1   (ya estamos en zona «encendida») 1
Lim. Derecha limt→2+ H(t) = 1   (seguimos en zona «encendida») 1
¿Coinciden? 1 = 1   ✅ LIMITE EXISTE = 1
✅ Teorema verificado: Como limt→2− H(t) = 1 = limt→2+ H(t), el teorema confirma que el limite existe y vale 1. Lejos del punto de salto, H(t) se comporta como una funcion constante y su limite se puede calcular por sustitucion directa.
💡 Conclusion: La funcion de Heaviside solo pierde su limite en el punto exacto del escalon (t = 0). En cualquier otro punto, como es constante a trozos, el limite existe trivialmente. ¡El teorema funciona perfectamente para identificar ambos casos!

✏️ Ejemplos Resueltos

Ejemplo 1:  Evaluar limt→0 H(t) ▼
Paso 1 — Punto de discontinuidad
La funcion H(t) tiene su punto de «encendido» en t = 0. Debemos evaluar los limites laterales alli.
Paso 2 — Limite por la izquierda
Cuando t → 0− (t se acerca a 0 por valores negativos), estamos en la zona donde H(t) = 0:
limt→0− H(t) = 0
Paso 3 — Limite por la derecha
Cuando t → 0+ (t se acerca a 0 por valores positivos), estamos en la zona donde H(t) = 1:
limt→0+ H(t) = 1
❌ Conclusion
Como 0 ≠ 1, el limite NO existe. Hay una discontinuidad de salto de magnitud 1 en t = 0.
Ejemplo 2:  Graficar H(t − 2) y evaluar limt→2 H(t−2) ▼
Paso 1 — Desplazamiento
H(t − 2) es la funcion escalon desplazada 2 unidades a la derecha. El «encendido» ocurre en t = 2.
Paso 2 — Grafica
y = 0 para t < 2 (linea horizontal sobre el eje t)
y = 1 para t ≥ 2 (linea horizontal a altura 1)
Circulo abierto en (2, 0), circulo cerrado en (2, 1).
Paso 3 — Limites laterales en t = 2
limt→2− H(t−2) = 0      limt→2+ H(t−2) = 1
❌ Conclusion
0 ≠ 1 ⇒ El limite NO existe. Discontinuidad de salto en t = 2.
Ejemplo 3:  Expresar f(t) = {0 si t<1, 3 si 1≤t<4, 0 si t≥4} con Heaviside ▼
Paso 1 — Analizar la funcion
La funcion «enciende» el valor 3 en t = 1 y lo «apaga» en t = 4. Necesitamos dos funciones escalon.
Paso 2 — Encendido en t = 1
Para encender el valor 3 en t = 1 usamos: 3 · H(t − 1)
3 · H(t−1) = 0 si t < 1,   3 si t ≥ 1
Paso 3 — Apagado en t = 4
Para apagar, restamos otro escalon: −3 · H(t − 4)
3·H(t−1) − 3·H(t−4)
✅ Resultado Final
f(t) = 3[ H(t − 1) − H(t − 4) ]
Esta expresion compacta reemplaza toda la definicion por partes.

🎯 Practica Interactiva

Puntaje: 0 / 3
1. ¿Cual es el valor de limt→0+ H(t)?
✅ Correcto! Para t ligeramente mayor que 0, H(t) = 1. El limite lateral por la derecha es 1.
❌ Incorrecto. Pista: Para valores de t justo mayores que 0, ¿en que zona de H(t) estamos?
💡 Solucion: Cuando t → 0+, estamos en la zona t > 0, donde H(t) = 1.
2. Expresa f(t) = {0 si t<2,  5 si t≥2} usando la funcion de Heaviside:
✅ Correcto! f(t) = 5·H(t−2) enciende el valor 5 en el instante t = 2.
❌ Incorrecto. Pista: Necesitas «encender» el valor 5 en t=2. Usa H(t−a) desplazada.
💡 Solucion: f(t) = 5H(t − 2). El escalon se desplaza a t=2 y se multiplica por 5.
3. ¿En que punto tiene discontinuidad H(t − 3)?
✅ Correcto! La discontinuidad esta en t = 3, donde el argumento t−3 se hace cero.
❌ Incorrecto. Pista: La discontinuidad de H(t) esta en t=0. Si desplazamos por 3...
💡 Solucion: H(t−3) tiene discontinuidad donde t−3 = 0, es decir en t = 3.

🌎 Aplicaciones en el Mundo Real

⚡

Circuitos electricos

Modelar el encendido de un circuito: un voltaje que se activa en el instante t = 0 y permanece constante.

V(t) = V0 · H(t)
⚗️

Sistemas de control

Las funciones de transferencia en automatizacion usan entradas escalon para analizar la respuesta de un sistema.

entrada(t) = A · H(t − t0)
📈

Modelado de eventos

Terremotos, dosis de medicamentos, cambios subitos de precio: eventos instantaneos modelados como escalones.

efecto(t) = magnitud · H(t − tevento)
