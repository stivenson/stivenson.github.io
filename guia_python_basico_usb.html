Sección 0 — Variables, Tipos y Entrada/Salida

En esta sección aprenderás los ingredientes básicos de cualquier programa: variables, tipos de datos (int, float, str), cómo pedirle datos al usuario con input() y cómo mostrar resultados formateados con print() y f-strings.

Conceptos clave:

  • int(input("...")) — leer un número entero
  • float(input("...")) — leer un número decimal
  • f"texto {variable}" — f-string para incrustar variables en texto
  • f"{valor:.2f}" — mostrar decimal con 2 cifras
  • f"{valor:<20}" — alinear texto a la izquierda en 20 caracteres

Sección 1 — Condicionales y Decisiones

Los condicionales permiten que el programa tome caminos distintos según los datos que recibe. Son el equivalente a "si pasa esto, haz aquello; si no, haz esto otro".

Conceptos clave:

  • if condicion: — ejecuta el bloque si la condición es verdadera
  • elif otra_condicion: — evalúa otra condición si la anterior fue falsa
  • else: — ejecuta si ninguna condición anterior fue verdadera
  • and, or, not — combinar condiciones
  • variable.strip().lower() — limpiar entrada del usuario (quitar espacios, convertir a minúsculas)

# Validador de pedido de domicilio (estilo Rappi)

# --- Entrada de datos ---
saldo_disponible = float(input("Saldo disponible ($): "))

distancia_ok = distancia_km <= 10
hora_ok     = 7 <= hora_actual <= 23

pedido_procede = saldo_ok and distancia_ok and hora_ok

    print("Pedido aprobado. Tu domicilio está en camino.")

    print("Pedido rechazado. Motivo(s):")
    if not saldo_ok:
        faltante = costo_pedido - saldo_disponible
        print(f"  - Saldo insuficiente. Te faltan ${faltante:.2f}.")
    if not distancia_ok:
        print(f"  - Distancia fuera de cobertura ({distancia_km} km). El máximo es 10 km.")

Ejercicio 1.5 — Calculadora de Descuento en Tienda 🏷️

Enunciado: Crea un programa que calcule el precio final de una compra acumulando hasta tres descuentos distintos: cupón (10%), membresía (5%) y compra por volumen de 3 o más unidades (8%).

Entradas:
- Precio original del producto en pesos (número decimal)
- ¿Tienes cupón de descuento? (s/n)
- ¿Eres miembro del programa de fidelidad? (s/n)

Salida esperada:

Descuento aplicado     : 23%
Precio unitario final  : $38,500.00
Total (3 und)          : $115,500.00
Ahorro total           : $34,500.00

💡 Pista: Acumula el porcentaje de descuento en una variable antes de calcular el precio final. El precio final unitario es precio_original × (1 - descuento / 100).

# Calculadora de descuento en tienda online

# --- Entrada de datos ---
precio_original = float(input("Precio original del producto ($): "))
tiene_cupon     = input("¿Tienes cupón de descuento? (s/n): ").strip().lower()
es_miembro      = input("¿Eres miembro del programa de fidelidad? (s/n): ").strip().lower()
cantidad        = int(input("Cantidad de unidades: "))

# --- Acumulación de descuentos ---
descuento_total = 0

if tiene_cupon == 's':
    descuento_total += 10   # 10% por cupón

if es_miembro == 's':
    descuento_total += 5    # 5% adicional por membresía

if cantidad >= 3:
    descuento_total += 8    # 8% adicional por volumen

# --- Cálculo de precios ---
precio_unitario_final = precio_original * (1 - descuento_total / 100)
precio_total_final    = precio_unitario_final * cantidad
ahorro_por_unidad     = precio_original - precio_unitario_final
ahorro_total          = ahorro_por_unidad * cantidad

# --- Resultados ---
print()
print(f"Descuento aplicado     : {descuento_total}%")
print(f"Precio unitario final  : ${precio_unitario_final:,.2f}")
print(f"Total ({cantidad} und)  : ${precio_total_final:,.2f}")
print(f"Ahorro total           : ${ahorro_total:,.2f}")

if descuento_total == 0:
    print()
    print("Sin descuentos aplicados. Usa un cupón o únete al programa de fidelidad para ahorrar.")

🧠 Cómo pensar antes de programar: Entradas → Proceso → Salidas

📌 Nota fundamental — lee esto antes de cada ejercicio

Antes de escribir una sola línea de código, todo problema debe descomponerse en tres partes:

📥 ENTRADAS

¿Qué datos necesita el programa para funcionar?
Son los valores que el usuario proporciona (input()) o que el programa recibe.
Ejemplo: el peso y la altura del usuario.

⚙️ PROCESO

¿Qué cálculos o decisiones hay que hacer con esos datos?
Son las fórmulas, condicionales, ciclos y llamadas a funciones.
Ejemplo: calcular imc = peso / altura² y clasificarlo.

📤 SALIDAS

¿Qué resultado debe mostrar el programa?

import turtle

# Configuración de la ventana
pantalla = turtle.Screen()
pantalla.title("Polígonos Regulares")
pantalla.bgcolor("white")

lapiz = turtle.Turtle()
lapiz.speed(5)
lapiz.pensize(2)

# Polígonos a dibujar: (número de lados, color, nombre en español)
poligonos = [
    (3, "red",    "Triángulo"),
    (4, "blue",   "Cuadrado"),
    (5, "green",  "Pentágono"),
    (6, "orange", "Hexágono"),
    (8, "purple", "Octágono"),
]

separacion = 150   # distancia horizontal entre figuras

x_inicio = -(len(poligonos) - 1) * separacion / 2

# Título en la parte superior
lapiz.penup()
lapiz.goto(0, 220)
lapiz.color("black")
lapiz.write("Polígonos Regulares", align="center", font=("Arial", 16, "bold"))

# Dibujar cada polígono
for i, (n, color, nombre) in enumerate(poligonos):
    angulo_exterior = 360 / n

    x = x_inicio + i * separacion
    y = -40  # posición vertical base

    lapiz.goto(x, y)
    lapiz.setheading(0)
    lapiz.color(color)
    lapiz.pendown()

    # Dibujar el polígono

        lapiz.forward(lado)
        lapiz.left(angulo_exterior)

    # Etiquetar debajo de la figura
    lapiz.penup()
    lapiz.goto(x + lado / 2, y - 25)
    lapiz.color("black")
    lapiz.write(nombre, align="center", font=("Arial", 10, "normal"))

# Ocultar la tortuga al terminar
lapiz.hideturtle()

def longitud_valida(contrasena):
    """Verifica si la contraseña tiene al menos 8 caracteres."""
    return len(contrasena) >= 8

def tiene_mayuscula(contrasena):
    """Verifica si la contraseña tiene al menos una letra mayúscula."""
    return any(c.isupper() for c in contrasena)

def tiene_numero(contrasena):
    """Verifica si la contraseña tiene al menos un dígito numérico."""
    return any(c.isdigit() for c in contrasena)

def tiene_especial(contrasena):
    """Verifica si la contraseña tiene al menos un carácter especial."""
    return any(c in "!@#$%^&*" for c in contrasena)

def nivel_seguridad(contrasena):
    """Determina el nivel de seguridad según cuántos criterios cumple."""
    criterios_cumplidos = sum([
        longitud_valida(contrasena),
        tiene_mayuscula(contrasena),
        tiene_numero(contrasena),
        tiene_especial(contrasena),
    ])

    if criterios_cumplidos == 1:
        nivel = "Débil"
    elif criterios_cumplidos == 2:
        nivel = "Media"
    elif criterios_cumplidos == 3:
        nivel = "Fuerte"
    else:

def calcular_porcentaje(votos, total):
    """Calcula el porcentaje de votos de un candidato."""
    if total == 0:
        return 0.0
    return (votos / total) * 100

def generar_barra(votos, max_votos, largo=20):
    """Genera una barra ASCII proporcional al máximo de votos."""
    if max_votos == 0:
        return "░" * largo
    llenos = round((votos / max_votos) * largo)
    vacios = largo - llenos
    return "█" * llenos + "░" * vacios

def mostrar_resultados(nombres, votos):
    """Muestra la tabla de resultados con barras ASCII y porcentajes."""
    total = sum(votos)
    max_votos = max(votos)
    nombre_mas_largo = max(len(n) for n in nombres)

    print()
    print("=== RESULTADOS DE VOTACIÓN ===")
    for i in range(len(nombres)):
        barra = generar_barra(votos[i], max_votos)
        porcentaje = calcular_porcentaje(votos[i], total)

    """Retorna el mes en que se alcanza el doble de seguidores iniciales."""
    meta = iniciales * 2
    for mes, cantidad in enumerate(lista_seguidores):
        if cantidad >= meta:
            return mes
    return None  # No se alcanzó el doble en el período simulado

def graficar_crecimiento(lista_seguidores):
    """Muestra una gráfica de línea con la proyección de crecimiento."""
    meses = list(range(len(lista_seguidores)))

    plt.figure(figsize=(10, 6))
    plt.plot(meses, lista_seguidores, color="royalblue", linewidth=2,
             marker="o", markersize=5, label="Seguidores")

    # Línea del doble inicial
    doble = lista_seguidores[0] * 2
    plt.axhline(y=doble, color="orange", linestyle="--", linewidth=1.5,
                label=f"Meta: {doble:,} (doble inicial)")

    plt.title("Proyección de Crecimiento de Seguidores", fontsize=14, fontweight="bold")
    plt.xlabel("Mes", fontsize=11)
    plt.ylabel("Seguidores", fontsize=11)
    plt.grid(True)
    plt.legend()
    plt.tight_layout()
    plt.show()
